## Running the script

### Without args

```shell
dart bin/dart_script.dart
```

### With args

args:

- `-a` = `--author-id`
- `-d` = `--depth`

```shell
dart bin/dart_script.dart -a Hstas4zQbsyXeRpAtzwjTZx37XiZL7PQGzWpo3b3jcSWsVW -d 1
```
